# hubot-chatbot

talk and meid

See [`src/chatbot.coffee`](src/chatbot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-chatbot --save`

Then add **hubot-chatbot** to your `external-scripts.json`:

```json
[
  "hubot-chatbot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
